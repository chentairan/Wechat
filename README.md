<h1><b>程序开发：</b></h1>

<h2>现版本（V0.9.7）存在的问题：</h2>
<p>
1. 首页（first）网络请求不稳定.可能原因:服务器不稳定。<br/>
2. 意见反馈页面反馈成功后，提示时间过短。解决方法：增加延时时间。<br/>
3. 关于我们页面可滑动。解决办法：修改样式。<br/>
4. 首页查询到信息后，再次打开小程序信息不会消失，解决办法：暂无。<br/>
5. 首页切换视图按钮触发区域过小，解决方法：修改view大小。<br/>
6. 完善样式信息，加强代码可读性。<br/>
待更新...
</p>
